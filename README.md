# Sonik

[WIP] Supersonik web framework.

## Quick Start

Just try it.

```
npx degit yusukebe/sonik/examples/basic my-app
cd my-app
```

npm:

```
npm install
npm run dev
npm run build
npm run deploy
```

yarn:

```
yarn install
yarn dev
yarn build
yarn deploy
```

## Preact preset

A Preact preset is available.

```
npx degit yusukebe/sonik/examples/preact preact-app
cd preact-app
```
