# Nashi.js

Nashi means "nothing". An almost nothing framework.

## Quick Start

Just try it.

```
npx degit yusukebe/nashi/examples/basic my-app
cd my-app
```

npm:

```
npm run install
npm run dev
npm run deploy
```

yarn:

```
yarn install
yarn dev
yarn deploy
```
