# Sonik

[WIP] Supersonik web framework for the Edges.

## Quick Start

Just try it.

```
npx degit yusukebe/sonik/packages/examples/basic my-app
cd my-app
```

npm:

```
npm run install
npm run dev
npm run deploy
```

yarn:

```
yarn install
yarn dev
yarn deploy
```
